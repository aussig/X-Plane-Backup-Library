# This X-Plane Scenery Library is a collection of almost 1000 water towers of many types, sizes, and colors.
# This library's primary purpose is for use in Headwind Hesket 'Grain Land' scenery packages soon to be available in the .org in early 2020.
# This libary package and it's (very large) companion package, 'Headwind Grain Land Library', is required in the 'Grand Land' scenery packages coming soon.
#    The 'Headwind Grain Land Libary' contains (about 5000) objects created for use in libary packages for areas in rural Kansas and surrounding areas.
#    This Grain Land Library contains sub-directories for Farms, Towns, and Cities; Grain Land Industry; Grain Storage; and Wichita and Wichita State University.
#    The 'Headwind Grain Land Libary' should be ready for upload to X-Plane.org in Janurary, 2020 and the associated Scenery soon after.
# The 'Headwind Water Tower Library' was kept separate from the Grain Land stuff because their a Water Towers all over the world and others might want to use them.
# This scenery library collection will by made available for download by anyone in the X-Plane.org download section.
# These water tower library objects and associated texture file may be used and/or altered freely by anyone for non-commercial use.
# Credit for creation of these objects (and textures) is not required but would by nice.  Commerical users contact my for permission. (HeadwindH@cox.net).

# Textures files
#
# ALL objects in this libary use texture files created by this author for use in these objects.  No texture file downloaded from the 3D Warehouse or anywhere else is included in this library.
#    Objects for X-Plane created by Sketch2XPlane require that only one texture file is allowed.  Virtually all .skp files in the 3D Warehouse contain several texture files.
#    Many of the 3D objects available for download for use by the public contain textures downloaded with 3D objects from Google Earth and other Google apps.
#    Texture files from Google are copyrighted by Google or by there many providers and are not allowed to be distributed for publice use in places like X-Plane.org.
#    As a result of the Sketchup2XPlane requirement for only one texture file AND to avoid copyright issues for texture files used in 3D Warehouse .skp files,
#    every original object in this HWH Water Tower library and every open sourse object download from other sources uses a texture file created by this author.
#    Anyone may use and/or modify any of the approximately 300 texture files included in this library package for non-commercial use.
#    Most of the textures are either 256x256 pixel .png files with two colored squares, or 512x512 pixel files with four colored squares.
#    There are a few larger files (such as 1024x1024) with contain more colored squares to give more color options to an object.
#    Most of the objects created by this author are simple shapes and simply textured and ARE NOT highly detailed but from a few hundred feet away look pretty good, IMHO  :) .
#         Hint: To easily change the color of an object, use a texture file with the same color squares pattern as the original, change the colors in a
#         graphics editor like Paint Shop, Photo Shop, Paint etc. and save the .png file with the same filename, or better yet, use a different filename.
#         If a different filename is used, edit the desired .obj file with a simple text editor like Notepad and change the texture filename in the .obj text to the new filename.
#    Most of the textures have a companion texture that end in _LIT.png. (i.e. Red-White.png and Red-White_LIT.png).  If the object's .obj text file contains a header line 
#    after the Texture name line (i.e. TEXTURE filename.png that reads TEXTURE_LIT filename_LIT.png) X-Plane will determine the darkness due to time of day and blend the
#    normal daylight texture with the _LIT texture to give some light to thes object.  Warning, the _LIT files should be very dark with almost nothing visible except
#    the lighted areas such as a window or light fixture.  Even those areas should be very dim or the object will glow far too brightly.
#    If the object headers (example below) contains a TEXTURE_LIT filename line follow tht TEXTURE line and no _LIT file is present the TEXTURE_LIT line will be ignored.
#
# 	800
# 	OBJ
#
# 	TEXTURE			HWWT_Stucco-Metal.png
# 	TEXTURE_LIT	 	HWWT_Stucco-Metal_LIT.png
#
# 	POINT_COUNTS	4224 0 0 10866
#
# ...
#
