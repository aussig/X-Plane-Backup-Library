# Headwind H (HWH) Radio Tower Library for X-Plane 10 and 11, v1.0 November 25, 2020
# This X-Plane Scenery Library is a collection of almost 500 radio, tv, and cellular towers of many types, sizes, colors and lighting.
# This library is a stand-alone library for those who wish to add tall lighted towers to their custom scenery projects.
# This library in available for free download in X-Plane.org Scenery Library section.
# This library's is a sub-set of radio tower objects in the Headwind H 'Grain_Land_Library' scenery library available in X-Plane.org.
# If the 'Grain_Land_Library' is already installed in your X-Plane Custom Scenery folder this library is not necessary.
# If it is desired for creators to add radio/tv/cellular tower object to scenery without installing the very large 4300 object Headwind Library
# than this Radio Tower Library might be preferred.  It is free to use by anyone for non-commercial purposes without permission or credit. 
# The 'Headwind Water Tower Library' was kept separate from the Grain Land stuff because there are Water Towers all over the world
# and others might want to use the water towers in their own scenery in parts of the world outside of Grainland in Central U.S.A.
# The same goes for this 'Headind Radio Tower Library.'
# All of these radio/tv/cellular objects and of their texture files were created by me, 'Headwind H' in X-Plane.org.
# The freeware version of Sketchup 2014 was used to create the 3D objects. 
# The great Sketch2XPlane Sketchup addon, downloaded from X-Plane.org was used to create the .obj files from Sketchup for use in X-Plane.
# Paint Shop Pro version was used to create the texture files used in the library.  There are only .png files.  Two are _LIT textures.
#
